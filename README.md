# SmartCart Clustering System

## Overview
SmartCart Clustering System is a machine learning project that performs customer segmentation using unsupervised learning techniques. By analyzing customer demographics and purchasing behavior, the system identifies distinct customer groups to support targeted marketing and business decision-making.

## Features
- Segmented 2,200+ customer records using clustering techniques.
- Performed data preprocessing including missing value treatment, outlier handling, encoding, and feature scaling.
- Engineered features such as Customer Tenure, Age, Family Size, and Total Spending.
- Applied Principal Component Analysis (PCA) for dimensionality reduction.
- Determined the optimal number of clusters using Elbow Method, Knee Detection, and Silhouette Score.
- Implemented and compared K-Means and Agglomerative Clustering algorithms.
- Visualized customer segments and analyzed cluster characteristics.
